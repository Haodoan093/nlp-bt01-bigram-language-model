BT02 - Evaluation of Two POS Taggers on the Brown Corpus
Run date: 2026-03-14

Dataset:
- Corpus: Brown (NLTK)
- Tagset: universal
- Evaluation setup: Brown is used only as the test set
- Number of test sentences: 57,340

Results:
1) PerceptronTagger (pretrained)
   - Precision (macro): 0.8241
   - Recall (macro):    0.8237
   - Macro-F1:          0.8205
   - Accuracy:          0.9184

2) BigramTagger (trained on Treebank)
   - Precision (macro): 0.8003
   - Recall (macro):    0.7532
   - Macro-F1:          0.7688
   - Accuracy:          0.8606

Conclusion:
- Brown was not used for training, only for testing.
- In this setup, the pretrained PerceptronTagger outperforms BigramTagger trained on Treebank.
